
Простой CLI.

Работает по следующему принципу.

Сначала строка поступает в метод exec_line класса line_executer.
Там сначала проверяется, не происходит ли в ней присваивания переменной.
Это делает метод create_variable класса dollar_handler.
Если есть знак =, слева от которого только буквы или цифры, то текст слева от = берётся за название переменной,
выражение справа вычисляется в виде строки и запоминается в map. Переходим к следующей строке.

Если присваивания переменной в строке нет, то дальше идёт попытка выполнить все подстановки вместо включений вида $...
Это делает метод replace_dollars класса dollar_handler.

Далее, метод get_parsed_lists класса parser разбивает строку по пайпам, после каждого | выделяет название команды и список её аргументов.

Наконец, далее в методе exec_line класса line_executer поочерёдно выполняются распарсенные команды, выход очередной команды передаётся
в качестве строки следующей, если они разделены пайпом.

Результат последней команды в строке выводится на экран.

Некоторые ошибки обрабатываются бросанием исключений и их отлавливанием.
